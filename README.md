# FarmaGreen 2.0 DNA

Nuevo backend + nuevo frontend local. No es parche de V1.5/V1.7 y no se deploya.

Inspiración conceptual: catálogos comerciales como Disfit, pero adaptado a FarmaGreen: sin carrito, sin checkout, consulta directa por WhatsApp.

## Rutas

- `/catalogo`
- `/producto/:slug`
- `/api/catalog`

## Local

```bash
npm install
npm run verify
PORT=8094 npm run dev
```
